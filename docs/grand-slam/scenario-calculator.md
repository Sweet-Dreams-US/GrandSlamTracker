# Scenario Calculator

## Projection Templates and Examples

Use this document to model fee projections for prospective and existing clients.

---

## How to Use

1. **Determine baseline** - Verify current annual revenue
2. **Identify category** - Match to size tier (Micro, Small, Medium, etc.)
3. **Estimate growth** - Conservative, target, and aggressive scenarios
4. **Run the scenario** - Fill in template or reference pre-built examples
5. **Evaluate viability** - Does it meet our $1,000/month minimum?

---

## The Three-Part Fee Structure Recap

### Year 1 Formula
```
Total Fees = Foundation Fee + Growth Fees
```

### Year 2+ Formula
```
Total Fees = Foundation Fee + Sustaining Fee + Growth Fees
```

Where:
- **Foundation Fee** = Annual Baseline × Category Rate
- **Growth Fees** = Sum of (Uplift per Tier × Tier Rate)
- **Sustaining Fee** = Last Year Avg - (New Foundation ÷ 12)

**Step 2: Calculate Growth Percentage**

**Step 3: Break Uplift into Tiers**

## Sustaining Fee Calculation Worksheet

### Step-by-Step Process

**Step 1: Calculate Last Year's Average Monthly Fee**
```
Total Year 1 Fees ÷ 12 = Average Monthly
$__________ ÷ 12 = $__________
```

**Step 2: Calculate New Baseline**
```
New Baseline = Old × Industry Factor × (1 + Growth% × Retention%)
New Baseline = $_____ × _____ × (1 + _____% × _____%)
New Baseline = $__________
```

**Step 3: Calculate New Foundation Fee**
```
New Foundation = New Baseline × Category Rate
New Foundation = $__________ × _____% = $__________
Monthly Foundation = $__________ ÷ 12 = $__________
```

**Step 4: Calculate Sustaining Fee**
```
Sustaining = Last Year Avg - New Monthly Foundation
Sustaining = $__________ - $__________ = $__________
```

### Worked Example

**Client:** Medium category, $500k baseline, achieved 80% growth, Year 1 avg fee $2,500/mo

**Step 1:** Last Year Avg = $2,500/month

**Step 2:**
```
New Baseline = $500,000 × 1.08 × (1 + 0.80 × 0.35)
New Baseline = $500,000 × 1.08 × 1.28
New Baseline = $691,200
```

**Step 3:**
```
New Foundation = $691,200 × 2.0% = $13,824/year
Monthly Foundation = $13,824 ÷ 12 = $1,152
```

**Step 4:**
```
Sustaining = $2,500 - $1,152 = $1,348/month
```

**Year 2 Floor:** $1,152 + $1,348 = **$2,500/month** (before any new growth)

### Rule of Thumb

```
If (Baseline × 0.10) > $1,000/month → Likely viable without minimum
If (Baseline × 0.10) < $1,000/month → Requires minimum fee or skip
```

- Baseline > $120k/year ($10k/mo) = Likely viable
