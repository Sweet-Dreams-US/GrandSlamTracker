# BASELINE SYSTEM

## Initial Baseline Calculation

When a client joins:

```
InitialBaseline = TrailingAverage × (1 + IndustryGrowthFactor)
```

**Example:**
- Client's trailing 12-month average: $25,000
- Industry: Remodeling (12%)
- Initial Baseline: $25,000 × 1.12 = $28,000

---

## Monthly Baseline (Seasonal Adjustment)

Each month, adjust baseline for seasonality:

```
MonthlyBaseline = AnnualBaseline × SeasonalIndex[month]
```

## Baseline Recalculation (Rolling Baseline)

At agreed intervals (typically annual), baseline recalculates:

```
NewBaseline = OldBaseline × (1 + IndustryGrowth) × (1 + RetainedGrowth)
```

**Retained Growth Options:**

| Option | They Keep | Formula |
|--------|-----------|---------|
| Conservative | 25% | AvgGrowth × 0.25 |
| Moderate | 35% | AvgGrowth × 0.35 |
| Aggressive | 50% | AvgGrowth × 0.50 |

**Example (Year 1 → Year 2):**

Year 1:
- Starting Baseline: $28,000
- Average Growth Achieved: 400%
- Foundation Fee: $28k × 2% = $560/month

Year 2 (Moderate retention):
- Industry Growth: 12%
- Retained Growth: 400% × 0.35 = 140%
- New Baseline: $28,000 × 1.12 × 2.40 = $75,264
- **New Foundation Fee: $75,264 × 2% = $1,505/month** (we got a raise!)

**What happens:**
- Our Foundation Fee nearly tripled
- Performance Fee now calculated on growth above $75,264
- We're incentivized to keep pushing for MORE growth
- We never start at $0

# FEE CALCULATION LOGIC FOR CLAUDE CODE

## Main Calculation Function

```javascript
function calculateMonthlyFee(client, month, actualRevenue) {
  
  // STEP 1: Get baseline for this month
  const baseline = calculateBaseline(client, month);
  
  // STEP 2: Get business size category
  const category = client.businessSizeCategory;
  
  // STEP 3: Calculate Foundation Fee
  const foundationRate = getFoundationFeeRate(category);
  const foundationFee = baseline * foundationRate;
  
  // STEP 4: Calculate uplift
  const uplift = actualRevenue - baseline;
  
  // STEP 5: Calculate Performance Fee
  let performanceFee = 0;
  let growthPercentage = 0;
  let tierBreakdown = [];
  
  if (uplift > 0) {
    growthPercentage = uplift / baseline;
    const tiers = getPerformanceFeeTiers(category);
    
    for (const tier of tiers) {
      if (growthPercentage > tier.growthFloor) {
        
        let applicableAmount;
        if (growthPercentage >= tier.growthCeiling) {
          // Full tier applies
          applicableAmount = (tier.growthCeiling - tier.growthFloor) * baseline;
        } else {
          // Partial tier
          applicableAmount = (growthPercentage - tier.growthFloor) * baseline;
        }
        
        const tierFee = applicableAmount * tier.feePercentage;
        performanceFee += tierFee;
        
        tierBreakdown.push({
          tier: tier.tierNumber,
          growthRange: `${tier.growthFloor * 100}% - ${tier.growthCeiling * 100}%`,
          applicableUplift: applicableAmount,
          feePercentage: tier.feePercentage,
          tierFee: tierFee
        });
      }
    }
  }
  
  // STEP 6: Calculate total
  const totalFee = foundationFee + performanceFee;
  
  return {
    month: month,
    actualRevenue: actualRevenue,
    baseline: baseline,
    uplift: Math.max(uplift, 0),
    growthPercentage: Math.max(growthPercentage, 0),
    
    foundationFee: foundationFee,
    foundationFeeRate: foundationRate,
    
    performanceFee: performanceFee,
    performanceBreakdown: tierBreakdown,
    
    totalFee: totalFee,
    effectiveRateOnRevenue: totalFee / actualRevenue,
    effectiveRateOnUplift: uplift > 0 ? performanceFee / uplift : 0
  };
}
```

## Baseline Calculation

```javascript
function calculateBaseline(client, month) {
  
  // Get base annual value
  let base;
  if (client.baselineMethod === 'trailing12') {
    base = average(client.trailingRevenue.slice(-12));
  } else if (client.baselineMethod === 'trailing6') {
    base = average(client.trailingRevenue.slice(-6));
  } else if (client.baselineMethod === 'custom') {
    base = client.customBaseline;
  }
  
  // Apply industry growth factor
  const industryFactor = getIndustryGrowthFactor(client.industry);
  let annualBaseline = base * (1 + industryFactor);
  
  // Check for baseline recalculation
  if (isRecalculationDue(client, month)) {
    annualBaseline = recalculateBaseline(client, annualBaseline);
    saveNewBaseline(client, annualBaseline);
  }
  
  // Apply seasonal adjustment
  let monthlyBaseline;
  if (client.seasonalAdjustment) {
    const seasonalIndex = getSeasonalIndex(client.industry, month);
    monthlyBaseline = annualBaseline * seasonalIndex;
  } else {
    monthlyBaseline = annualBaseline;
  }
  
  return monthlyBaseline;
}

function recalculateBaseline(client, currentBaseline) {
  
  // Get average growth delivered
  const avgGrowthRate = calculateAverageGrowthRate(client);
  
  // Get retention factor
  let retainedGrowth;
  if (client.growthRetentionOption === 'conservative') {
    retainedGrowth = avgGrowthRate * 0.25;
  } else if (client.growthRetentionOption === 'moderate') {
    retainedGrowth = avgGrowthRate * 0.35;
  } else if (client.growthRetentionOption === 'aggressive') {
    retainedGrowth = avgGrowthRate * 0.50;
  }
  
  // Apply industry growth + retained growth
  const industryFactor = getIndustryGrowthFactor(client.industry);
  const newBaseline = currentBaseline * (1 + industryFactor) * (1 + retainedGrowth);
  
  return newBaseline;
}
```

# SUMMARY

## The System

| Component | Purpose |
|-----------|---------|
| **Foundation Fee** | Guarantees we never earn $0, grows as baseline grows |
| **Performance Fee** | Rewards aggressive growth, 8 tiers up to 1000%+ |
| **Rolling Baseline** | Recalculates periodically, gives us raises |

## Growth Tiers (8 Tiers)

| Tier | Growth | Multiplier |
|------|--------|------------|
| 1 | 0-50% | Up to 1.5x |
| 2 | 51-100% | 1.5x to 2x |
| 3 | 101-200% | 2x to 3x |
| 4 | 201-300% | 3x to 4x |
| 5 | 301-500% | 4x to 6x |
| 6 | 501-750% | 6x to 8.5x |
| 7 | 751-1000% | 8.5x to 11x |
| 8 | 1001%+ | Beyond 11x |

## Key Numbers

- Micro business ($5k) grown to $50k (1000%) = **~$6,925/month fee**
- Small business ($15k) grown to $90k (500%) = **~$9,975/month fee**
- Medium business ($50k) grown to $250k (400%) = **~$19,750/month fee**

**We get paid when we deliver transformational growth.**
